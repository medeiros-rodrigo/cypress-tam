# Script de Teste

Scripts de testes em cucumber/gherkin-syntaxed specs com Cypress.io



## Instalação Cypress
Instalar dependências do Cypress  
```bash
npm install -D cypress
```

## Execução dos Testes
* Abrir Cypress `npm run open-cy`
* Selecionar teste `Passagem.feature`
